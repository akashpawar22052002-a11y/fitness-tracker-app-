# 🏋️‍♂️ Fitness Tracker Application

This is a Full-Stack **Fitness Tracker Application** designed to help users log, track, and manage their daily workout routines efficiently. The application is built entirely using the **MERN Stack** (MongoDB, Express.js, React, Node.js).

---

## 📁 Project Structure

The project has been organized according to the submission guidelines into the following directory structure:

*   **`backend/`** : Contains the Node.js, Express.js, and MongoDB REST API logic.
*   **`frontend/`** : Contains the React.js user interface (UI) and design components.
*   **`screenshot/`** : Contains task screenshots, visual proofs, and related submission documents.

---

## ✨ Features

### 🖥️ Backend (API Layer)
*   **Full CRUD Operations**: Complete ability to Create, Read, Update, and Delete workout logs.
*   **Robust Logging**: Integrated server-side logging mechanism to track API requests, responses, and errors.
*   **Input Validation**: Strict request payload validation to ensure clean data formatting before database insertion.

### 🌐 Frontend (User Interface)
*   **Responsive Dashboard**: Clean, intuitive, and mobile-friendly dashboard interface.
*   **Dynamic Workout Form**: Easy-to-use form to quickly log new activities and exercises.
*   **Interactive List**: View all previously logged workouts in a structured layout with instant deletion support.
